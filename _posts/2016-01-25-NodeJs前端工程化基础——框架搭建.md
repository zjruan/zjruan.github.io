官网： [https://nodejs.org/en/](https://nodejs.org/en/)

Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node.js 使用了一

全球最大的开源库生态系统。
